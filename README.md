# AIアジャイルTDDプロジェクト開発フレームワーク
このリポジトリは、AIにプロジェクト開発を委託するための最小構成テンプレートです。アジャイル+TDDを厳守し、GitHub Actionsでテストを絶対実行。ソフトウェアエンジニア/AIエンジニア向けに設計され、人間レベルの品質管理を目指します。

## 🚀 クイックスタート
### このリポジトリをフォーク/クローン。
``` bash
git clone https://github.com/kodai-n11qbb/ai2code.git
cd ai2code
```
- /mds/REQUIREMENTS.md を編集（要件・禁止技術を人間記述）。
- AI（ChatGPT/Claudeなど）に下記プロンプトで実装してもらう。
```
/mds/REQUIREMENTS.md を読んで行動してください
```
